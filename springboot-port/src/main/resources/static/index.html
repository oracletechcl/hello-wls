Hello Spring Boot!

Host Information Application (Migrated from WebLogic)

Welcome to the Spring Boot 3.x demonstration application. This is a modernized version of the WebLogic Server 12.2.1.4 application with REST APIs, embedded Tomcat, and HikariCP connection pooling.
View Host Information (JSON) Database Connection (JSON) Session Management (JSON) Greeting Service (JSON) Swagger API Docs

Application Features:

  • REST APIs with JSON responses (not HTML servlets)
  • Spring Boot 3.x with embedded Tomcat
  • Host information API (hostname, IP, OS, Java, memory)
  • Database connectivity with HikariCP (instead of Oracle UCP)
  • HTTP session management with Spring Session support
  • Spring @Service beans (instead of EJB @Stateless)
  • RESTful endpoints (instead of JAX-WS SOAP)
  • Dependency injection with @Autowired (instead of @EJB)
  • Interactive Swagger/OpenAPI documentation
  • Spring Boot Actuator for health checks
  • Fast startup time (~5 seconds vs 60+ seconds)
