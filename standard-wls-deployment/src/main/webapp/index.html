Hello WebLogic!

Host Information Application

Welcome to the WebLogic Server 12.2.1.4 demonstration application. Explore server environment, database connectivity, session management, and EJB web services.
View Host Information Database Connection Session Management EJB Web Service

Application Features:

  • Network Information (Hostname, IP Address, Network Interfaces)
  • Operating System Details (Name, Version, Architecture)
  • Java Runtime Information (Version, Vendor, Home Directory)
  • Memory Statistics (Max, Total, Free, Used Memory)
  • User & Environment Information
  • Oracle Autonomous Database Integration (Mock & Real ADB)
  • Connection Pool Management with Oracle UCP
  • Database Connectivity Testing & Monitoring
  • HTTP Session Management & Tracking
  • Session Persistence & Replication (Clustered HA)
  • Session State Management & Timeout Control
  • Stateless Session Beans (EJB 3.x)
  • JAX-WS SOAP Web Services
  • Dependency Injection with @EJB
  • Automatic WSDL Generation

Deployed on Oracle WebLogic Server 12.2.1.4
